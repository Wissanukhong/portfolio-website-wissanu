Courses undertaken

คอร์สเรียนต่างๆ ที่ผมได้เข้าเรียนรู้ ศึกษาและพัฒนาตัวเองให้เป็น Web developer

1.คอร์สเรียนออนไลน์สอนเขียนเว็บไซต์ 0-100

หลังจากที่ผมเข้าเรียนคอร์สนี้ทำให้เข้าใจหลักการทำงานของเว็บไซต์ว่าทำงานอย่างไร จะต้องเขียนโค้ดอย่างไร ขั้นตอนในการออกแบบเว็บไซต์ ขั้นตอนการเขียนโค้ด รวมถึงการวางแผน ในการพัฒนาเว็บไซต์, Tester, debug code, การวิเคราะห์ปัญหาและการแก้ไขปัญหา เรียนรู้การใช้งานเครื่องมือและ framwork ต่างๆ เช่น VScode, Git version control, Bitbucket, Bootstrap4, Firebase hosting

2.คอร์สเรียนออนไลน์เริ่มต้นเขียนเว็บไซต์ด้วย HTML5&CSS3

ได้เรียนรู้หลักการทำงานของเว็บไซต์เบื้องต้น ว่าเว็บไซต์คืออะไร ทำงานอย่างไร เข้าใจหลักการทำงานของ Tag ต่างๆ ใน HTML5 และการตกแต่งเว็บไซต์ด้วย CSS3 ซึ่งทำให้เข้าใจหลักการพัฒนาเว็บไซต์ ด้วยภาษา HTML5 & CSS3 ในเบื้องต้น

3.คอร์สเรียนออนไลน์ Programming for Everyone

เรียนรู้วิทยาการคอมพิวเตอร์เบื้องต้นว่าคอมพิวเตอร์คืออะไร ทำงานอย่างไร มีการแปลงภาษาเครื่องแบบไหน และได้เรียนรู้พื้นฐานต่างๆ ในการเขียนโปรแกรม ไม่ว่าจะเป็น ขั้นตอนการเขียนโปรแกรม อัลกอริทึม ขั้นตอนการทำงานของโปรแกรม Flowchart การวิเคราะห์ปัญหา แนวทางการแก้ไขปัญหา ฯลฯ ซึ่งทำให้เข้าใจหลักการทำงานของโปรแกรมมากยิ่งขึ้น และสามารถนำความรู้ที่ได้ไปต่อยอดในกาเขียนโปรแกรมในระดับต่อไป

4.พื้นฐานที่ควรรู้ก่อนสร้างเว็บ ด้วย HTML5 & CSS3

เรียนรู้พื้นฐานการทำงานของเว็บไซต์ และภาษาที่ใช้ในการพัฒนาเว็บไซต์เบื้องต้น ทำให้เข้าพื้นฐานว่าเว็บไซต์ทำงานอย่างไร ฝึกการลงมือทำเว็บไซต์ โดยประยุกต์ใช้ความรู้ที่ได้จากคอร์สเรียน ผ่าน Workshop ต่างๆ

5.BASIC-PHP-PROGRAMMING

หลังจากที่ผมได้เข้าเรียนรู้คอร์สนี้ ผมได้รับความรู้เกี่ยวกับภาษา PHP มากยิ่งขึ้น ได้เรียนรู้หลักการทำงานของเว็บไซต์ และการพัฒนาเว็บไซต์โดยใช้ ภาษา PHP และได้เรียนรู้ว่าเว็บไซต์ทำงานอย่างไร ภาษา PHP ทำงานอย่างไร มีความสำคัญอย่างไรต่อการพัฒนาเว็บไซต์ ซึ่งจะทำให้เราสามารถนำความรู้ที่ได้ ไปต่อยอดอะไรได้บ้าง ทำให้เห็นภาพรวมการทำงานของภาษา PHP ผ่านการฝึกลงมือทำผ่าน Workshop ต่างๆ ที่ทางสมาคมโปรแกรมไทยได้จัดเตรียมไว้

6.Programmer for non-programmer

หลังจากที่ได้เข้าเรียนคอร์ส Programming for non programmimg ทำให้ได้เรียนรุ้โครงสร้างพื้นฐานต่างๆ ในการเขียนโปรแกรม ไม่ว่าจะเป็น หลักการ Coding หรือว่า Algorithm แนวทางการแก้ไขปัญหา ถ้าเราเจอปัญหาแบบนี้ จะแก้ไขปัญหาอย่างไร แนวทางการฝึกฝนการเป็นโปรแกรมเมอร์ จาก Zero - Hero ซึ่งจะทำให้เรามีพัฒนาการในการคิด การแก้ไขปัญหา อัลกอริทึม ได้มากยิ่งขึ้น

7.Career & personality traits for programmers

หลังจากที่เข้าเรียนคอร์ส Career & personality traits for programmers ทำให้ได้ความรู้เกี่ยวกับการทำงานในสายงานโปรแกรมเมอร์ ว่า เกี่ยวข้องกับอะไรบ้าง มีทักษะอะไรบ้างทีเราต้องเรียนรู้ โปรแกรมเมอร์ทำงานอย่างไร แบ่งออกเป็นกี่ประเภท อะไรบ้าง จะพบเจอปัญหาอะไรบ้างในการทำงาน และจะต้องแก้ไขปัญหานั้นอย่างไร และได้เรียนรู้แนวทางการแก้ไขปัญหา การคิด ผ่าน workshop ที่ทางสมาคมโปรแกรมไทยได้จัดเตรียมไว้ และที่สำคัญจะทำให้เรารู้ว่า เราถนัดทางด้านไหน และจะต้องพัฒนาต่อยอดอย่างไร

8.อบรมเรื่อง BIG DATA ที่บริษัท ธนูลักษณ์ จำกัด (มหาชน)

หลักสูตรอมรมเรื่อง Big data ที่จัดขึ้นระหว่างที่ผมทำงานอยู่ที่บริษัท ธนูลักษณ์ จำกัด (มหาชน) ทำให้ผมได้เรียนรู้พื้นฐาน การวิเคราะห์ข้อมูล เข้าใจหลักการทำงานของ Big data และวิทยาศาสตร์ข้อมูล การนำข้อมูลนั้นมาทำการวิเคราะห์ สังเคราะห์ และการนำข้อมูลนั้นไปประโยชน์
